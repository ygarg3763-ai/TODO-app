# TODO-app
A simple Todo List application built using HTML, CSS, and JavaScript.

## Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Responsive Design
- Data stored using localStorage

## Technologies Used

- HTML
- CSS
- JavaScript
- DOM Manipulation
- Local Storage

## Project Structure

```
Todo-List-App
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run

1. Download the project.
2. Open `index.html` in your browser.
3. Start managing your tasks.
